# Professional Dashboard

This is a comprehensive cross-platform Flutter application featuring a professional metrics dashboard with beautiful charting capabilities.

## Product Overview

The Professional Dashboard is designed for business leaders, analysts, and project managers who need to monitor key performance indicators in real-time. It provides an at-a-glance summary of vital operational metrics alongside robust historical comparisons using interactive charts.

## Key Features
- **Metrics Overview:** Quick-access summary cards for Total Revenue, Active Users, Churn Rate, and Server Costs with dynamic percentage change indicators.
- **Revenue Performance Chart:** A beautiful, interactive grouped bar chart displaying current vs previous period revenue comparisons across months.
- **Responsive Design:** Optimized grid layout that dynamically adapts from single-column on mobile devices to multi-column arrangements on wide desktop monitors.
- **Interactive Tooltips:** Touch/hover over the chart to see precise underlying values.

## Tech Stack
- Flutter ^3.7.2
- `fl_chart`: For high-performance, customizable and interactive data visualizations.

## Setup Instructions

1. Ensure you have the Flutter SDK installed on your machine.
2. Clone this repository.
3. Fetch the required dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application on your preferred target (Web, iOS, Android, macOS, Windows, Linux):
   ```bash
   flutter run
   ```
